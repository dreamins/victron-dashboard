# Gemini Project Instructions

## General Rules
* **Testing:** ALWAYS add new automated tests (unit or UI) with every code change or feature implementation. Never assume a change is correct without verification.

## Git Workflow
* **Commit Tagging:** When making commits to this repository, always include `[Gemini]` or `gemini:` at the start of the commit message to help the user distinguish between changes made by human, Claude, and Gemini.
